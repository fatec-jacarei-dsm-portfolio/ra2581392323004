## 🎨 Design & Tecnologia

O portfólio foi construído com **HTML, CSS e JavaScript** puros, sem frameworks, com foco em:

- **Dark mode** com paleta `#0d0d14` e accent `#7c3aed` (violeta)
- **Glassmorphism** nos cards com `backdrop-filter: blur()`
- **Fluxo contínuo** entre seções (sem caixas separadas)
- **Tipografia fluida** com `clamp()` e escala baseada em `vw` — responsivo de 375px a 5944px
- **Timeline de experiência** com linha vertical animada
- **Carrossel** horizontal com `scroll-snap` para projetos e publicações
- **section-label** em todas as seções para hierarquia visual clara
