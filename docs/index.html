5º Semestre — Mentora — Plataforma Inteligente de Apoio Acadêmico

Cargo: Desenvolvedor Full Stack  |  Equipe: InnoCodeSolutions  |  Período: 2025/2

Projeto: Mentora — Plataforma de Machine Learning para Apoio Acadêmico

O Mentora utiliza Machine Learning para prever desempenho acadêmico e risco de evasão. Alunos preenchem formulários sobre hábitos de estudo, sono, frequência e motivação pelo app mobile; modelos de Regressão Linear, Regressão Logística e Random Forest processam esses dados e retornam previsões personalizadas com explicações via SHAP.

Minhas contribuições individuais:

  • Desenvolvimento do backend em Node.js + TypeScript: rotas de autenticação, coleta de dados e integração com os modelos de ML
  • Modelagem e implementação do banco de dados em PostgreSQL via Prisma ORM
  • Implementação do pipeline de integração entre o backend e os modelos Python (Scikit-learn, SHAP)
  • Desenvolvimento de telas do aplicativo mobile em React Native (formulários de hábitos e tela de resultados)
  • Configuração de pipelines de CI/CD e containerização com Docker
  • Implementação do dashboard do professor com visão de alunos em risco de evasão

Tecnologias: React Native · Node.js · TypeScript · Python · Scikit-learn · SHAP · PostgreSQL · Prisma · JWT · Docker · CI/CD

6º Semestre — ProconBot Jacareí Em Desenvolvimento

Cargo: Desenvolvedor Full Stack  |  Equipe: InnoCodeSolutions  |  Período: 2026/1

Projeto: ProconBot Jacareí — Chatbot Inteligente de Defesa do Consumidor

O ProconBot é um chatbot inteligente acessível via WhatsApp que auxilia consumidores a obter orientações sobre seus direitos. A base de conhecimento é composta pelas FAQs do PROCON e pelo Código de Defesa do Consumidor (CDC), combinando fluxos decisórios estruturados com técnicas de PLN e RAG para interpretação e resposta às mensagens.

Minhas contribuições individuais:

  • Implementação do motor de fluxo decisório baseado nas orientações do PROCON
  • Desenvolvimento da integração com o WhatsApp via whatsapp-web.js e gestão de sessões de conversa
  • Implementação da persistência de histórico de mensagens (US11) com PostgreSQL
  • Estruturação e ingestão da base de conhecimento FAQ do PROCON para o pipeline de RAG
  • Geração de embeddings e implementação de busca semântica com pgvector
  • Configuração da infraestrutura em nuvem e pipelines de CI/CD
  • Desenvolvimento de logs de auditoria das interações

Tecnologias: Node.js · TypeScript · whatsapp-web.js · RAG · PLN · LLM · PostgreSQL · pgvector · Docker · CI/CD

Modeling Multi-Sensor Daily Fire Events in Brazil: The DescrEVE Relational Framework for Wildfire Monitoring

Revista: Remote Sensing — MDPI  |  Ano: 2026  |  Vol.: 18, Nº 4, Artigo 606

DOI: 10.3390/rs18040606


Autores: Henrique Bernini, Fabiano Morelli, Fabrício Galende Marques de Carvalho, Guilherme dos Santos Benedito, William Max dos Santos Silva, Samuel Lucas Vieira de Melo.


Artigo científico publicado na revista internacional Remote Sensing (Qualis A1 / JCR Q1), desenvolvido como parte das atividades de Iniciação Científica no INPE. O trabalho apresenta o framework relacional DescrEVE, que integra dados de múltiplos satélites para modelar eventos diários de fogo no Brasil, gerando descritores estruturados por bioma e região para substituir o modelo tabular vigente na plataforma TerraBrasilis do INPE.
