Resumo

Sou estudante em Desenvolvimento de Software pela FATEC e atualmente curso o 4° Semestre e atuo como bolsista de Iniciação Científica.


Tenho amplo experiência no desenvolvimento de software para o ambito de produtos tecnológicos e pesquisa ciêntifica, obstinado por aprender novas técnologias e conhecimento para aplicar ao mundo real com o objetivo de transformar e solucionar problemas. Tenho fascinio por psicologia, agro e computação, onde aplico meus conhecimentos com o objetivo transformador de otimizar o simples para escalar em soluções complexas.


Minha trajetória é guiada propósito de aliar o desenvolvimento e inocação ao um ambiente acolhedor gerando soluções inovadoras e transformando vidas.

Programa Queimas DescrEVE

INPE - Instituto Nacional de Pesquisas Espaciais
Função: Desenvolvedor de Software - Biomas BR CNPq

Iniciação Científica

Técnicas de Processamento de Imagens e Inteligência Artificial Aplicadas à Detecção e Descrição de Área Queimada.
Pesquisador de Iniciação Ciêntifica - CNPq

O Programa Queimas DescrEVE é uma iniciativa do INPE voltada para o monitoramento de queimadas. Este projeto processa imagens do satélite e depois gera estátitica, fornecendo dados de Descritor e Evento de Fogo para monitorar incêndios no Brasil.

Minha colaboração: Desenvolvedor de Software do DescrEVE.
Contribuições: Desenvolvimento de software para Geração de Descritor e responsável por criação de query e API's em um arquitetura de microserviços para disponibizar por meio de API's informações de Descritor de Fogo por Região e Evento de Fogo por Região para todo o Brasil.
