Segundo Semestre

Projeto: Sistema Web para Manipulação de Projetos de Mapeamento

Projeto desenvolvido por alunos do 2º semestre do curso de Desenvolvimento de Software Multiplataforma onde atuei como Desenvolvedor de Software, na FATEC Professor Francisco Moura em Jacareí.
O objetivo foi criar uma aplicação web para gerenciamento e manipulação de projetos de mapeamento, com foco em eficiência no armazenamento e organização dos dados.

O sistema oferece funcionalidades para registrar, editar e visualizar projetos de forma estruturada, atendendo às demandas de organização e controle em ambientes acadêmicos ou corporativos.

As tecnologias utilizadas incluem TypeScript no desenvolvimento da aplicação e PostgreSQL, como sistema gerenciador de banco de dados com a extensão PostGIS para gestão de dados geográficos e a interface for feito em HTML.

Terceiro Semestre

Projeto: Aplicativo de Monitoramento Nutricional

Projeto desenvolvido por alunos do 3º semestre do curso de Desenvolvimento de Software Multiplataforma onde atuei como Desenvolvedor de Software, na FATEC Professor Francisco Moura em Jacareí, com foco na proposta de empreendedorismo.
O objetivo principal do aplicativo é monitorar a ingestão diária de calorias e nutrientes, oferecendo suporte ao usuário para o controle da alimentação e a manutenção de uma vida saudável e equilibrada.

A solução combina funcionalidades práticas com uma interface intuitiva, permitindo o acompanhamento contínuo da dieta e dos hábitos alimentares.

As tecnologias utilizadas incluem Node.js, TS-Node.js, MongoDB, TypeScript, React, além do uso de Trello e Kanban para organização e gerenciamento do projeto.

Quarto Semestre

Projeto: Aplicativo de Monitoramento via Dispositivo IoT

Esse projeto foi desenvolvido pelos alunos do 4º semestre do curso de Desenvolvimento de Software Multiplataforma onde atuei como Desenvolvedor de Software da FATEC Professor Francisco de Moura, em Jacareí, com foco na proposta de empreendedorismo.
O aplicativo tem como objetivo monitorar derivadores. Para isso, antes do desenvolvimento da aplicação, foi criado um dispositivo IoT responsável pela coleta e comunicação dos dados com o sistema.

A integração entre hardware e software proporciona uma solução completa para o monitoramento em tempo real, utilizando tecnologias modernas e multiplataforma.

As tecnologias utilizadas incluem Node.js, TS-Node.js, MongoDB, TypeScript, React, React Native, Android, iOS, C++, além do uso de Trello e Kanban para organização e gerenciamento do projeto.

Iniciação Científica

Apresentação da prova de conceito para um sistema descritor da ocorrência de fogo nos biomas brasileiros. Desenvolvido como parte de uma Iniciação Científica, o trabalho explora o uso de tecnologias de processamento de imagens de satélite para gerar descritores compreensíveis, com o objetivo de substituir modelos tabulares existentes na plataforma do INPE.
