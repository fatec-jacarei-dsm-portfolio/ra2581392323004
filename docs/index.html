Quinto Semestre

Projeto: Mentora — Plataforma Inteligente de Apoio Acadêmico

Projeto desenvolvido pelos alunos do 5º semestre do curso de Desenvolvimento de Software Multiplataforma, onde atuei como Desenvolvedor de Software, na FATEC Professor Francisco de Moura em Jacareí.
O Mentora é uma plataforma que utiliza Machine Learning para prever o desempenho acadêmico e o risco de evasão de estudantes. A solução coleta dados sobre hábitos de estudo, frequência, sono e engajamento dos alunos por meio de um aplicativo mobile em React Native, processando-os com modelos de Regressão Linear, Regressão Logística e Random Forest.

O sistema oferece painéis dedicados para Alunos (previsões e feedback explicativo via SHAP), Professores (dashboard com alunos em risco) e Instituições (métricas agregadas por curso e disciplina).

Tecnologias utilizadas:

  • React Native (mobile)
  • Node.js · TypeScript
  • Python (Machine Learning — Scikit-learn, SHAP)
  • PostgreSQL · Prisma ORM
  • JWT · Docker · CI/CD

Sexto Semestre Em Desenvolvimento

Projeto: ProconBot Jacareí — Chatbot de Defesa do Consumidor

Projeto em desenvolvimento pelos alunos do 6º semestre do curso de Desenvolvimento de Software Multiplataforma, onde atuo como Desenvolvedor de Software, na FATEC Professor Francisco de Moura em Jacareí.
O ProconBot é um chatbot inteligente acessível via WhatsApp que auxilia consumidores a obter orientações sobre seus direitos. O sistema combina fluxos decisórios baseados nas orientações do PROCON com técnicas de PLN e RAG (Retrieval Augmented Generation), utilizando as FAQs do PROCON e o Código de Defesa do Consumidor como base de conhecimento.

A solução registra e audita todas as interações, gera orientações claras e direciona o usuário ao próximo passo, com infraestrutura hospedada em nuvem.

Tecnologias utilizadas:

  • Node.js · TypeScript
  • whatsapp-web.js (integração WhatsApp)
  • RAG · PLN · LLM
  • PostgreSQL · pgvector
  • Infraestrutura em Nuvem · CI/CD

Modeling Multi-Sensor Daily Fire Events in Brazil: The DescrEVE Relational Framework for Wildfire Monitoring

Revista: Remote Sensing — MDPI  |  Ano: 2026  |  Vol.: 18, Nº 4, Artigo 606

DOI: 10.3390/rs18040606


Autores: Henrique Bernini, Fabiano Morelli, Fabrício Galende Marques de Carvalho, Guilherme dos Santos Benedito, William Max dos Santos Silva, Samuel Lucas Vieira de Melo.


Artigo científico publicado na revista internacional Remote Sensing (Qualis A1 / JCR Q1), desenvolvido como parte das atividades de Iniciação Científica no INPE. O trabalho apresenta o framework relacional DescrEVE, que integra dados de múltiplos satélites para modelar eventos diários de fogo no Brasil, gerando descritores estruturados por bioma e região para substituir o modelo tabular vigente na plataforma TerraBrasilis do INPE.
